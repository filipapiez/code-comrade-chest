Spec Guide

Five things to spec early

Get these set with your GC and IT team before walls are framed.

System

Stick-built vs demountable

Stick-built is cheapest and permanent. Demountable systems unbolt and move; modular kits ship from a manufacturer.

  • Stick-built: lowest cost, permanent
  • Demountable: relocatable, premium
  • Modular: branded systems, fast install
Privacy

Clear, frosted or smart glass

All three can live on the same wall. Frosted bands at sitting height are the most common privacy treatment.

  • Clear with manifestation film
  • Frosted privacy band
  • Switchable smart glass for premium offices
Doors

Sliding, swing or pivot

Sliding doors save floor space; swing doors fit code and ADA more easily; pivot doors are the design flex.

  • Top-hung sliding on a barn track
  • Swing with concealed closer
  • Pivot door for executive offices
Acoustics

Single vs double glazing

Single-glazed partitions are visual only. Double glazing with sealed perimeter gives real conference-room acoustics.

  • Single glazed: ~28–32 STC
  • Double glazed: ~35–40 STC
  • Door seals matter as much as glass
Install

Coordinate with the GC

Floor levelness, ceiling deflection and electrical/data routing all need to be agreed before glass is fabricated.

  • Floor flat within ⅛″ over 10′
  • Ceiling deflection head detail
  • Power/data in adjacent stud walls
